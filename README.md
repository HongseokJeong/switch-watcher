# switch-watcher: Watch Nintendo Switch's stock in Online Stores

This script checks Nintendo Switch's stock in online shops. If stock is available, Open shop's URL on your browser automatically.

- Node.js v7.6.0 ~
- Chrome 59 ~

## How to Use

```
$ npm i
$ npm start
```
